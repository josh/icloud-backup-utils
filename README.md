# icloud-backup-utils

Scripts creating backups of iCloud data.

## Apps

- [x] iCloud Drive
- [ ] Photos
- [ ] Mail
- [ ] Contacts
- [ ] Calendar
- [ ] Reminders
- [x] Notes
- [x] Keychain

## Support

- [x] macOS Mojave 10.14
- [x] macOS Catalina 10.15
- [ ] macOS Big Sur 11

## Install

```sh
$ brew install josh/tap/icloud-backup-utils
```
